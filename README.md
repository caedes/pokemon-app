# Pokemon App

## Getting Started

Install dependencies:

```shell
yarn
```

Run the dev server:

```shell
yarn start
```

To deploy the app:

```shell
yarn build
```

then serve the `/dist` on [Netlify](https://www.netlify.com/).

## Folder Structure

A class should be in its own folder and wrritten in an index.js file. e.g.

```
/src
    /App
        index.js
```
